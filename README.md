# OpenClaw Sandbox

Safe testing environment for OpenClaw integrations and updates.

## What this is for

- Test OpenClaw updates before production
- Try new integrations safely
- Debug issues in isolation
- Experiment with configs

## Quick Start

1. Click "Code" ‚Üí "Open with Codespaces" 
2. Wait for environment to load (~60 seconds)
3. Test away!

## Security

This environment is completely isolated from James's production OpenClaw setup. Nothing here can affect the main system.

## Pre-installed

- Node.js 18+
- OpenClaw CLI (via postCreateCommand)
- GitHub CLI
- Common ports forwarded (3000, 8080, 9000)
